# Controle de Estoque

Um sistema full-stack de gerenciamento de estoque com React (frontend) e Node.js (backend).

## Quick Start

### Frontend
cd frontend && npm install && npm run dev

### Backend
cd backend && npm install && docker compose up -d && npm run dev

## Stack

Frontend: React 19, TanStack Router, Tailwind, Vite
Backend: Express, Prisma, PostgreSQL, JWT
